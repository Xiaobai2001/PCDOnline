# PCD 在线编辑器

一个基于 Vue 3 + Three.js 的点云地图在线编辑工具，支持 PCD 文件的导入、编辑、可视化和管理。

## 功能特性

### 📁 文件管理

- **导入地图**：支持上传 `.pcd`、`.pgm`、`.yaml` 格式文件
- **本地存储**：导入的文件自动保存到 `pcd` 目录下持久化存储
- **地图列表**：显示本地导入和分组的地图信息
- **批量操作**：使用、编辑、导出、删除地图
- **分组管理**：支持创建文件分组，统一管理 PCD/PGM/YAML 文件

### ✏️ 点云编辑工具

| 功能 | 说明 |
|------|------|
| **移动** | 平移画布视图 |
| **点云绘制** | 使用圆刷在指定位置添加点云 |
| **点云擦除** | 使用圆刷擦除点云数据 |
| **框选点云擦除** | 框选区域批量删除点云（带二次确认弹窗） |
| **圆刷半径** | 可调节绘制/擦除的圆刷大小 (0.05m - 5m) |

### 🗺️ 栅格地图编辑

| 功能 | 说明 |
|------|------|
| **添加栅格** | 在指定区域标记为占用 |
| **擦除栅格** | 清除指定区域的栅格占用 |
| **框选擦除栅格** | 框选区域批量清除栅格 |
| **显示/隐藏** | 切换栅格图层可见性 |

### 💾 文件操作

- **保存修改**：保存当前编辑内容
- **另存为...**：将当前地图另存为新文件
- **提交到服务器**：将编辑后的地图提交到后端服务器

### 🎨 UI 设计特点

- **小圆角风格**：现代化圆角设计语言
- **玻璃拟态效果**：半透明毛玻璃背景
- **渐变主题**：蓝紫色系渐变配色
- **响应式布局**：自适应不同屏幕尺寸
- **智能布局**：
  - 地图信息横向一行排布（名称 + 大小 + 点位数）
  - 操作按钮采用网格布局（2×2 或 3 列）
  - 信息过长时支持水平滚动查看
  - 操作按钮固定位置不动

## 技术栈

### 前端
- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全
- **Three.js** - 3D 渲染引擎（用于点云可视化）
- **Element Plus** - UI 组件库
- **Vite** - 构建工具

### 后端
- **ASP.NET Core** - Web API 框架
- **C#** - 编程语言

## 核心功能实现

### 1. 坐标系统

- **屏幕坐标 → 世界坐标**：自动转换鼠标点击位置到点云坐标系
- **Y 轴修正**：已修正 Y 轴反转问题，确保上下方向正确
- **相机偏移补偿**：支持平移后的准确坐标映射

### 2. 点云数据处理

- **空间索引**：高效的点云查询和操作
- **增量更新**：新增的点云可以正常擦除
- **实时渲染**：编辑操作即时反馈到画布

### 3. 多层渲染

- **点云层**：Three.js Canvas 渲染
- **栅格层**：HTML5 Canvas 2D 叠加
- **同步缩放**：两层保持一致的视图变换

### 4. 事件处理

- **鼠标事件**：mousedown / mousemove / mouseup 完整链路
- **滚轮事件**：平滑缩放
- **右键菜单**：预留扩展接口
- **工具模式切换**：可靠的按钮状态绑定

## 快速开始

### 安装依赖

```bash
cd frontend
npm install
```

### 启动开发服务器

```bash
# 启动前端
npm run dev

# 启动后端（另一个终端）
cd backend
dotnet run
```

### 访问应用

前端默认运行在 `http://localhost:5173`  
后端 API 运行在 `http://localhost:5000`

## 使用说明

1. **导入地图**：点击"导入地图"按钮，选择 `.pcd` 文件
2. **选择地图**：在左侧列表中点击"使用"或"编辑"
3. **编辑操作**：
   - 选择顶部工具栏的工具模式（移动/绘制/擦除等）
   - 调整圆刷半径参数
   - 在画布上进行编辑操作
4. **保存结果**：点击"保存"或"提交到服务器"

## 注意事项

- 支持的文件格式：`.pcd`（点云）、`.pgm`（栅格）、`.yaml`（配置）
- 文件大小限制：建议单个 PCD 文件不超过 100MB
- 浏览器兼容性：推荐使用 Chrome / Edge 最新版本
- 编辑历史：当前版本不支持撤销/重做，请及时保存
